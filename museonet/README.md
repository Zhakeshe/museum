# E-Museum.kz - TSX Conversion

Этот проект представляет собой конвертированную версию сайта e-museum.kz из HTML в TSX (TypeScript + React/Next.js).

## Структура проекта

```
src/
├── components/
│   ├── Header.tsx      # Компонент шапки сайта
│   └── Footer.tsx      # Компонент подвала сайта
├── pages/
│   └── index.tsx       # Главная страница
└── types/
    └── index.ts        # TypeScript типы и интерфейсы
```

## Особенности

- **Header Component**: Включает бургер-меню, навигацию, поиск, языковой переключатель
- **Footer Component**: Включает логотип, меню, контактную информацию, социальные сети
- **TypeScript**: Полная типизация компонентов и данных
- **Responsive Design**: Адаптивный дизайн для мобильных устройств

## Установка и запуск

Если вы используете Next.js:

```bash
npm install
npm run dev
```

Если вы используете Create React App:

```bash
npx create-react-app my-app --template typescript
# Скопируйте компоненты в src/
npm start
```

## Компоненты

### Header
- Бургер-меню для мобильных устройств
- Полноэкранное меню
- Навигация по сайту
- Поиск
- Языковой переключатель (Рус/Қаз/Eng)
- Ссылка на вход в систему

### Footer
- Логотип и копирайт
- Навигационное меню
- Контактная информация
- Социальные сети (Facebook, WhatsApp, Instagram)
- Кнопка "Связаться с нами"
